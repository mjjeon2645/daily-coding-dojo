---

### 1. 이해
- 노란색, 갈색 격자의 개수를 각각 알고 있음
- 카펫의 가로 길이는 세로 길이와 같거나 세로 길이보다 김
- 리턴되는 배열의 각 원소의 곱이 매개변수 둘의 합과 동일해야 함
